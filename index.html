Welcome to a provisional Flagmatic website.

The original website Flagmatic.org is down, so I set up this temporary page. The latest version of Sage that I know it runs on (and gives approximately correct answers) is Sage-7.3beta3 (so Sage-7.2 if you want a released version).

Installation

See the README file in the Github repository. It gives a short and hopefully working sequence of steps to get Flagmatic set up in Sage.

Documentation

You can find the original User Guide by Emil here. There is a User Guide there for the dev version of Flagmatic. In the dev user guide, ignore the installation instructions (they should work though).

Authors and Contributors
