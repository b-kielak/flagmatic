Welcome to Flagmatic website.

This is a provisional website for Flagmatic package written by Emil Vaughan. The original website Flagmatic.org is down, so I set up this temporary page for the package here. The latest version of Sage that I know it runs on (and gives approximately correct answers) is Sage-7.3beta3 (so Sage-7.2 if you want a released version).
